## Ниша

**AI-инструменты для бизнеса и самозанятых** — практические гайды, обзоры, инструкции.

**Почему выбрана:**
- RPM в РСЯ: 150–350 руб/1000 показов (B2B/Tech категория)
- Трафик: 8–12 млн/мес, рост +400%
- Партнёрки: SaaS recurring 20–30%/мес (Notion, Canva, Jasper, Zapier)
- Контент на русском — острая нехватка, слабые конкуренты

## Стек

- **CMS:** WordPress (REST API для публикации)
- **Генерация:** Python + Claude API (OpenRouter)
- **Агенты:** Researcher → Writer → Editor → SEO
- **Хостинг:** Selectel / TimeWeb / Beget
- **SEO:** Yoast SEO плагин
- **Analytics:** Яндекс.Метрика + Google Analytics

---

## Цикл генерации (из astro-agent)

```
Тема из content-plan.md
  → Researcher (Haiku) — собирает факты
  → Writer (Sonnet) — пишет статью
  → Editor (Sonnet) — правит
  → SEO (Haiku) — title, description, slug, теги
  → WordPress REST API — публикация
```
