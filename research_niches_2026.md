### 🥇 МЕСТО 1: Коды ошибок бытовой техники

**Суть:** Страницы вида «Стиральная машина [Марка] ошибка E2 — причины и решение»

| Критерий | Оценка | Комментарий |
|---|---|---|
| RPM | 7/10 | 80–200 руб/1000 показов (авто/техника) |
| Объём трафика | 9/10 | Миллионы запросов, каждая поломка = запрос |
| Автогенерация | 10/10 | Идеальный шаблон: марка + модель + код = статья |
| Конкуренция | 9/10 | Конкуренты — старые форумы с ужасным UX |
| Партнёрки | 8/10 | Запчасти: Exist.ru, Autodoc, Ozon (3–8%) |
| **ИТОГ** | **8.6/10** | |

**Почему идеально для astro-agent:**
- Шаблон 100% предсказуем: «Что означает ошибка → Причины → Решение → Когда звонить мастеру»
- Programmatic SEO: марка × модель × код ошибки = 500 000+ уникальных страниц
- Данные стабильны, не устаревают (коды ошибок не меняются)
- Картинки: Unsplash/Pexels (бытовая техника)

**Партнёрки:**
- Запчасти: Ozon (3–8%), Wildberries, Exist.ru, Autodoc
- Сервисные центры: CPA 300–1500 руб/лид
- РСЯ: стабильный CPM

**Примеры запросов:**
- «стиральная машина samsung ошибка E4» — 3 000/мес
- «холодильник Indesit мигает лампочка» — 5 000/мес
- «посудомойка bosch ошибка E15» — 8 000/мес
- Таких запросов — сотни тысяч

## ТОП-3 РЕКОМЕНДАЦИИ ПОД СИСТЕМУ ASTRO-AGENT

---

### ВАРИАНТ А: «Ремонт техники» (самый простой старт)

**Почему этот вариант:**
- Programmatic SEO в чистом виде — идеально под автогенерацию
- Шаблон универсальный: `[Марка] [Модель] ошибка [Код]`
- Сотни тысяч уникальных страниц из одного шаблона
- Минимальный риск YMYL-фильтров (не медицина, не финансы)
- Конкуренты — старые форумы без SEO-оптимизации

**Как адаптировать astro-agent:**
```
brief.md    → описание сайта по бытовой технике
tone.md     → технический, но понятный стиль
plan.md     → список: марки × модели × коды ошибок
Агент       → Ресёрчер ищет причины ошибки
            → Писатель пишет по шаблону
            → SEO заполняет мета
Картинки    → Unsplash (бытовая техника)
```

---

### ВАРИАНТ Б: «Самозанятость и ИП» (максимальный RPM)

**Почему этот вариант:**
- Финансовая ниша = самый дорогой RPM в РСЯ
- CPA партнёрки банков: 2000–10 000 руб за лид
- Аудитория 10+ млн самозанятых в РФ
- Статьи вечнозелёные (законы меняются редко)

**Как адаптировать:**
```
brief.md    → сайт для самозанятых и малого бизнеса
tone.md     → простой язык, объяснение без юридизмов
plan.md     → 250+ тем: регистрация, налоги, отчётность
Партнёрки   → Тинькофф, Контур, МоёДело через Admitad
```

---

### ВАРИАНТ В: «AI-инструменты» (быстрый рост + recurring)

**Почему этот вариант:**
- Самая быстрорастущая ниша в рунете

- Легко генерировать: обзоры инструментов по шаблону

**Как адаптировать:**
```
brief.md    → сайт об AI-инструментах для бизнеса
tone.md     → практический, без технических терминов
plan.md     → 250+ тем: обзоры, гайды, сравнения
Партнёрки   → Jasper, Canva, Notion через Admitad
```

## ПЛАН ДЕЙСТВИЙ

### Шаг 1: Выбрать нишу (1 день)
Ответить на вопрос: что проще всего шаблонизировать под твою систему?

### Шаг 2: Адаптировать astro-agent (3–5 дней)
- Переписать `brief.md` под новую нишу
- Переписать `tone-of-voice.md`
- Создать `content-plan.md` с 250+ темами
- Настроить партнёрки (Admitad + РСЯ)

### Шаг 3: Запустить и наполнить (1–3 месяца)
- 1 статья в день = 30 статей в месяц
- 3 месяца = 90 статей = начало органического трафика

### Шаг 4: Подключить монетизацию (после 500 посетителей/день)
- РСЯ: минимум 500 посетителей/сутки для подачи заявки
- Партнёрки: с первого дня

### Шаг 5: Масштабировать (6–12 месяцев)
- Добавить programmatic SEO компонент
- Нанять редактора для улучшения топ-20 статей
- Расширить партнёрские программы
